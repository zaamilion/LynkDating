# LynkDating

**LynkDating** — сервис онлайн-знакомств, реализованный на основе микросервисной архитектуры

## Stack
-   **FastAPI** - Python фреймворк для написания API, основной компонент бэкенда
- **PostgreSQL** - Хранение данных пользователей 
- **Redis** - Хранение кэшируемых данных
- **Kafka** - передача взаимодействий пользователей
- **aioboto3** - Работа с S3 хранилищем
-  **Docker** - Контейнеризация проекта
- **Git** - Управление версиями
- **nginx** - проксирование фронтенда и API Gateway
  
*Frontend: html + css + js*
